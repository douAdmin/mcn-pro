# [MCN 管理系统](https://wangbosire.github.io/mcn-pro)

## 安装依赖

安装 `node_modules`:

```bash
npm install
```
or
```bash
yarn install
```
or
```bash
cnpm install
```

## 脚本列表

详情见`package.json`里面`scripts`字段

### 项目启动

```bash
npm start
```

### dist打包

```bash
npm run build
```

### 检查代码样式

```bash
npm run lint
```

### 测试脚本

```bash
npm test
```

### 项目配置文件
根目录`config`文件夹是项目的配置文件

`src`里面的`config`文件夹是开发时的配置文件
